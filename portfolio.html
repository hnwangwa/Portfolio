MoCP.me

This Node Express web application uses computer vision as a way to interact with the Museum of Contemporary Photography (MoCP) collection on user-generated terms. Since 2012, the collection has been sorted into hashtags to provide students from all over Chicago better access to the online database - Using a MySQL database curated with help from the museum, we hope to expand on this platform by developing a visual tagging system.
